Projeto Dados dos Atletas - DevStart

Aplicação em JavaScript puro que utiliza Programação Orientada a Objetos (POO) para gerenciar dados de atletas e realizar cálculos de performance e saúde.

Sobre o Projeto

Este software utiliza uma classe chamada `Atleta` para encapsular informações pessoais e métodos de cálculo. É uma evolução do projeto de notas, integrando parâmetros físicos como peso e altura.

Funcionalidades

Cálculo de IMC: Baseado no peso e altura do atleta.
Categorização por Idade:
Infantil`: 9 a 11 anos.
Juvenil`: 12 e 13 anos.
Intermediário`: 14 e 15 anos.
Adulto`: 16 a 30 anos.
Média Válida: Cálculo que elimina a maior e a menor nota do júri.

Como Executar

Pré-requisitos
[Node.js](https://nodejs.org/) instalado.

Passo a passo
1. Abra o terminal na pasta do projeto.
2. Execute o comando:
bash
node notas-atletas2.js
